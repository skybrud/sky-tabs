# sky-tabs

A Vue component for creating tabs easily.

## Dependencies

_None_

## Usage

```
<sky-tabs>
	<sky-tabs-item>
		<p>Your content goes here.</p>
	</sky-tabs-item>
	<sky-tabs-item>
		<p>Your content goes here.</p>
	</sky-tabs-item>
	<sky-tabs-item>
		<p>Your content goes here.</p>
	</sky-tabs-item>
</sky-tabs>
```
